# Prompt Enhancement App (Next.js)

A minimal **Next.js + TypeScript** app with a simple password-protected landing page.

## Getting started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Auth flow

- Landing page: `/`
- Main app (protected): `/app`

A client-side cookie is set after successful password entry.

## Deploy

This project is deployable to Vercel as-is (default Next.js build settings).
